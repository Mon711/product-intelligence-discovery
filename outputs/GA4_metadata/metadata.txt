====================
DIMENSIONS
====================
API name: achievementId
Display name: Achievement ID
Description: The achievement ID in a game for an event. Populated by the event parameter `achievement_id`.

API name: adFormat
Display name: Ad format
Description: Describes the way ads looked and where they were located. Typical formats include `Interstitial`, `Banner`, `Rewarded`, and `Native advanced`.

API name: adSourceName
Display name: Ad source
Description: The source network that served the ad. Typical sources include `AdMob Network`, `Liftoff`, `Facebook Audience Network`, and `Mediated house ads`.

API name: adUnitName
Display name: Ad unit
Description: The name you chose to describe this Ad unit. Ad units are containers you place in your apps to show ads to users.

API name: appVersion
Display name: App version
Description: The app's versionName (Android) or short bundle version (iOS).

API name: audienceId
Display name: Audience ID
Description: The numeric identifier of an Audience. Users are reported in the audiences to which they belonged during the report's date range. Current user behavior does not affect historical audience membership in reports.

API name: audienceName
Display name: Audience name
Description: The given name of an Audience. Users are reported in the audiences to which they belonged during the report's date range. Current user behavior does not affect historical audience membership in reports.

API name: audienceResourceName
Display name: Audience resource name
Description: The resource name of this audience. Resource names contain both collection & resource identifiers to uniquely identify a resource; to learn more, see [Resource names](//google.aip.dev/122).

API name: brandingInterest
Display name: Interests
Description: Interests demonstrated by users who are higher in the shopping funnel. Users can be counted in multiple interest categories. For example, `Shoppers`, `Lifestyles & Hobbies/Pet Lovers`, or `Travel/Travel Buffs/Beachbound Travelers`.

API name: browser
Display name: Browser
Description: The browsers used to view your website.

API name: browserVersion
Display name: Browser version
Description: The version of the browser from which the user activity originated. For example, a version might appear as `146.0.7680.178`. To understand which browser this version belongs to, use this version alongside the Browser dimension.

API name: campaignId
Display name: Campaign ID
Description: The identifier of the marketing campaign. Present only for key events. Includes Google Ads Campaigns, Manual Campaigns, & other Campaigns.

API name: campaignName
Display name: Campaign
Description: The name of the marketing campaign. Present only for key events. Includes Google Ads Campaigns, Manual Campaigns, & other Campaigns.

API name: character
Display name: Character
Description: The player character in a game for an event. Populated by the event parameter `character`.

API name: city
Display name: City
Description: The city from which the user activity originated.

API name: cityId
Display name: City ID
Description: The geographic ID of the city from which the user activity originated, derived from their IP address.

API name: cm360AccountId
Display name: CM360 account ID
Description: The CM360 Account ID that led to the key event. Identifies the CM360 Account.

API name: cm360AccountName
Display name: CM360 account name
Description: The CM360 Account Name that led to the key event. A CM360 account consists of advertisers, sites, campaigns, and user profiles.

API name: cm360AdvertiserId
Display name: CM360 advertiser ID
Description: The CM360 Advertiser ID that led to the key event. A CM360 Advertiser contains a group of campaigns, creative assets, and other settings.

API name: cm360AdvertiserName

API name: cohort
Display name: Cohort
Description: The cohort's name in the request. A cohort is a set of users who started using your website or app in any consecutive group of days. If a cohort name is not specified in the request, cohorts are named by their zero based index such as cohort_0 and cohort_1.

API name: cohortNthDay
Display name: Daily cohort
Description: Day offset relative to the firstSessionDate for the users in the cohort. For example, if a cohort is selected with the start and end date of 2020-03-01, then for the date 2020-03-02, cohortNthDay will be 0001.

API name: cohortNthMonth
Display name: Monthly cohort
Description: Month offset relative to the firstSessionDate for the users in the cohort. Month boundaries align with calendar month boundaries. For example, if a cohort is selected with the start and end date in March 2020, then for any date in April 2020, cohortNthMonth will be 0001.

API name: cohortNthWeek
Display name: Weekly cohort
Description: Week offset relative to the firstSessionDate for the users in the cohort. Weeks start on Sunday and end on Saturday. For example, if a cohort is selected with the start and end date in the range 2020-11-08 to 2020-11-14, then for the dates in the range 2020-11-15 to 2020-11-21, cohortNthWeek will be 0001.

API name: comparison
Display name: Comparison
Description: Comparisons enable you to evaluate subsets of your data side by side. For report requests, the comparisons column is automatically added to the response. For pivot report requests, the requested dimension must be used.

API name: contentGroup
Display name: Content group

API name: countryId
Display name: Country ID
Description: The geographic ID of the country from which the user activity originated, derived from their IP address. Formatted according to ISO 3166-1 alpha-2 standard.

API name: currencyCode
Display name: Currency
Description: The local currency code (based on ISO 4217 standard) of the eCommerce event. For example, `USD` or `GBP`. Currency is specified in tagging by the `currency` parameter. Businesses that transact in more than one currency can specify a local currency code when sending eCommerce events to Analytics, and this dimension shows those currencies. To Learn more, See [Currency reference](https://support.google.com/analytics/answer/9796179).

API name: dayOfWeek
Display name: Day of week

API name: dayOfWeekName
Display name: Day of week name
Description: The day of the week in English. This dimension has values such as Sunday or Monday.
